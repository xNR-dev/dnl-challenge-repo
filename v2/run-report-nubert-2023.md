# Nubert 2023 Run Report

## Purpose

This document is a non-normative example run report for the Nubert electronic GmbH 2023 filing.

It exists to keep empirical outputs separate from the normative `v2` specification.

## Example metrics captured in a run report

- checklist item count
- verification outcome counts
- notable findings
- known extraction limits
- execution notes
- token or cost observations, if tracked

## Historical note

Earlier versions of `v2-spec.md` included point-in-time run metrics and cost estimates directly in
the specification. Those metrics have been moved out because they describe one run, not the
normative design.

## Suggested contents for a completed run report

1. Run metadata
2. Source manifest summary
3. Applicability/exemption summary
4. Checklist metrics
5. Verification metrics
6. Key findings
7. Known limitations
